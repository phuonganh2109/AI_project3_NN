* Put the dataset in the same folder with the source code.

* For Dogs and Cats dataset: 
	- Download from the Internet and keep the zip format file.
	- Run python file.
	- https://www.kaggle.com/c/dogs-vs-cats/data
* For MNIST: 
	- Download all zip files from the Internet.
	- Create folder named 'mnist' and put all dataset into that folder then execute code.
	- http://yann.lecun.com/exdb/mnist/
* For MNIST-Fashion dataset:
	- Download all zip files from the Internet.
	- Create folder named 'fashion' and put all dataset into that folder then execute code.
	- https://github.com/zalandoresearch/fashion-mnist/tree/master/data/fashion
